Dunder methods (also called special methods) in Python are methods with
double underscores before and after their names, such as init, str, and
add.

They allow developers to define how objects behave with built-in
operations.

Common Examples:

1.  init
    -   Constructor method
    -   Runs automatically when an object is created
2.  str
    -   Defines what gets printed when using print()
3.  add
    -   Defines behavior of + operator

Example Class:

class Number: def init(self, value): self.value = value

    def __str__(self):
        return f"Value is {self.value}"

    def __add__(self, other):
        return Number(self.value + other.value)

Test

n1 = Number(10) n2 = Number(20)

print(n1) print(n1 + n2)

Output: Value is 10 Value is 30
