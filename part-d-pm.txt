Explanation:

Normal Distribution: A normal distribution is a bell-shaped distribution
where most values are around the mean. It is defined by mean (μ) and
standard deviation (σ).

Z-score: Z-score measures how far a value is from the mean in terms of
standard deviations. Formula: Z = (X - μ) / σ

Hypothesis Testing: It is used to make decisions based on sample data. -
Null hypothesis (H0): No effect - Alternative hypothesis (H1): There is
an effect

Python Code:

import numpy as np

data = np.random.normal(50, 10, 100)

mean = np.mean(data) std = np.std(data)

z = (data[0] - mean) / std

print(“Mean:”, mean) print(“Std Dev:”, std) print(“Z-score:”, z)

mu = 50 n = len(data)

z_stat = (mean - mu) / (std / (n ** 0.5))

if abs(z_stat) > 1.96: print(“Reject H0”) else: print(“Fail to reject
H0”)
